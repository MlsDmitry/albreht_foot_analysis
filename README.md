Для работы с проектом удобно запускать jupyter notebook из корневой папки git-репозитория. Так как пути к изображениям, используемые в ipynb ноутбуках указаны относительно корнейвой папки. Пример запуска jupyter для работы с проектом:

```
jupyter notebook
```

Для работы удобно установить сразу все зависимости, используемые в рамках репозитория. Зависимости описаны в файле `pyproject.toml`
 
```
pip install .
```

Или с помощью пакетного менеджера для python [uv](https://github.com/astral-sh/uv)

```
uv sync
```
